# Install local

1. Git clone or download in zip file

```bash
git clone git@github.com:smurfov/MyTaskManager.git
```

2. Install dependecies

```bash
npm install
bun install
```

3. Dev mode

```bash
npm run dev
bun dev
```

4. Build mode

```bash
npm run prod
bun prod
```
