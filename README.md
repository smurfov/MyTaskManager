# Install local

1. You need the `bun`. This project not on the `npm`.

```bash
npm install -g bun
```

2. Git clone or download in zip file

```bash
git clone git@github.com:smurfov/MyTaskManager.git
```

3. Install dependecies

```bash
bun install
```

4. Dev mode

```bash
bun dev
```

5. Build mode

```bash
bun prod
```
